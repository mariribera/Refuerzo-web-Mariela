# 🖥️ Mantenimiento y Aseguramiento de la Operación del Sistema Informático

## Descripción del Proyecto

Página web informativa sobre el **mantenimiento y aseguramiento de la operación del sistema informático**, desarrollada como proyecto académico para documentar los conceptos fundamentales, tipos de mantenimiento, herramientas y mejores prácticas del área.

## Estructura del Proyecto

```
mantenimiento-sistema/
│
├── index.html          # Página principal con navegación por menús
├── style.css           # Estilos visuales de la página
└── README.md           # Documentación del proyecto (este archivo)
```

## Contenido de la Página

La página informativa incluye las siguientes secciones navegables:

1. **Inicio** — Presentación general del módulo
2. **Conceptos** — Definiciones y fundamentos del mantenimiento informático
3. **Tipos** — Mantenimiento preventivo, correctivo, predictivo y adaptativo
4. **Herramientas** — Software y herramientas utilizadas
5. **Procedimientos** — Pasos y protocolos estándar
6. **Contacto** — Información de contacto

## Tecnologías Utilizadas

- HTML5 semántico
- CSS3 con variables personalizadas (custom properties)
- JavaScript vanilla para la navegación y animaciones
- Git para control de versiones

## Cómo Visualizar

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/usuario/mantenimiento-sistema.git
   ```
2. Abrir `index.html` en cualquier navegador moderno.
