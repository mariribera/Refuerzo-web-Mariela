Mantenimiento del Sistema Informático

Guía completa sobre el mantenimiento preventivo, correctivo y predictivo de sistemas informáticos. Aprende herramientas, procedimientos y mejores prácticas del sector.

Explorar Contenido

Conceptos Fundamentales

Sistema Informático

Conjunto integrado de hardware, software y recurso humano que trabaja en coordinación para capturar, almacenar, procesar y distribuir información.

Mantenimiento Informático

Conjunto de actividades técnicas y administrativas orientadas a conservar, restaurar o mejorar el estado funcional de los componentes de un sistema informático.

Disponibilidad del Sistema

Porcentaje de tiempo durante el cual un sistema informático opera correctamente.

Aseguramiento de la Operación

Proceso que garantiza que todos los componentes del sistema informático funcionan dentro de los parámetros establecidos.

Tipos de Mantenimiento

Preventivo

Se ejecuta de forma planificada y periódica para evitar fallos futuros.

Correctivo

Se realiza después de que ocurre una falla para restaurar el funcionamiento.

Predictivo

Basado en el monitoreo continuo para anticipar fallos antes de que ocurran.

Adaptativo

Modificaciones realizadas para adaptar el sistema a nuevos entornos o requisitos.
